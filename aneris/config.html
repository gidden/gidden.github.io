Run Control and ConfigurationÂ¶

The full harmonization process can be configured and controlled using a RunControl (RC) yaml file. The default values used for all parameters are shown below.

config:
    default_luc_method: reduce_ratio_2150_cov
    cov_threshold: 20
    harmonize_year: 2015
prefix: CEDS+|9+ Sectors
suffix: Unharmonized
add_5regions: true
